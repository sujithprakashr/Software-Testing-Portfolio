# 📁 Test Sets

This folder contains exported test set reports for the nopCommerce eCommerce application, managed using Jira and Xray.

## 📌 Contents

- PDF exports of individual test sets
- Organized test cases grouped by functional modules

## 🎯 Purpose

Test Sets were created to logically group related test cases, enabling efficient test execution, coverage tracking, and regression testing. These documents demonstrate structured test organization and real-world test management practices.

## 🛠 Tool Used

- Jira
- Xray Test Management
